check-pkg - Check that required packages are in your package.json

  usage: check-pkg [options]

  options:
    -h, --help                    show help and usage
    -v, --version                 show version
    -f, --file-filter <filter>    set the file filter(s)
    -d, --dir-filter <filter>     set the directory filter(s)
    -e, --excludes <filter>       set the excludes filter(s)
    -q, --quiet                   only show errors
